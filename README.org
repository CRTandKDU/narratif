# narratif

* Narratif: Narrative interactive fiction
~Narratif~ is an Emacs Org-based tool for creating and reading interactive narrations. Readers navigate through the story-game by clicking on links.

~Narratif~ draws inspiration from toolsets like [[https://docs.textadventures.co.uk/squiffy/][Squiffy]], [[https://github.com/idmillington/undum][Undum]], or [[https://twinelab.net/index.html][Twine]] and many others used in text games and gameboooks development.
